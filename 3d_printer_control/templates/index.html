How It Works

Connect to Wi-Fi

Ensure your wireless module is connected to your Wi-Fi network.

Access the Web Interface

Open a web browser and enter the IP address of your Raspberry Pi or ESP module.

Upload Design Files

Go to the file upload section and upload your 3D design files.

Monitor Printing

Use the web interface to track the print progress and make adjustments as needed.

Start Print Job

Select the uploaded file, configure print settings, and start the print remotely.
